# Chat con Socket.io

Ejemplo de creación de un chat con [Socket.io](https://socket.io/) y [express.js](https://expressjs.com/).

## Scripts disponibles

Para la instalación de los paquetes componen a este proyectos ejecutar el comando:

```bash
npm install
```

Para ejecutar el proyecto

```Bash
npm run server
```
